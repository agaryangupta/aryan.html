First would be the elements in HTML


HTML elements are the building blocks of web pages. They consist of a start tag, content, and an end tag. Types of elements include:

  • Block-level elements: These take up the full width of the page (e.g., < div>, < p>, < h1>, < section>, < article>, < header>, < footer>).
  • Inline elements: These only take up as much width as necessary (e.g., < span>, < a>, < img>, < strong>, < em>, < b>, < i>).
